Bootcamp 2020: Days Gone

The very first project that i completed within my first year of University was an animation project that i had to complete within two weeks of the initial startup. This Processing project demonstrates a comprehensive understanding of the use of arrays, loops, and classes.

My project is a forest that changes depending on the position of the sun and the moon. There is also a bird that flies past in the beginning and it has an audio file that is attached to it. After some time it begins to rain and I created the rain through the use of an array and a class. i was also able to impliment an audio file of trees in the wind and rain when it begins to fall. If you look carefully you will also be able to see that the trees are actually growing every time that rains and one tree grows at a faster rate than the others. This project was the commencement of my programming journey and I hope every project after shows you how much my skills and knowledge has improved.

Check out the code on Github: https://github.com/HumzahC/adbt103.git
