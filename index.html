CarWay

CarWay is my very first fully functioning booking website that I created for another assignment I received in university. It is a car maintenance service that allows user to register into the website and create a booking for any service they wanted done to their car, whether it be an MOT, Tyre Change, Full Service or Brake check. The website would also show when your next appointment was and if the time for a new booking was available. And the best part was that everything was being stored onto a SQL Database so the website worked incredibly well but don’t just believe me, check it out for yourself and try telling me that it doesn’t look professional (apart from all of the llorum ipsum).
